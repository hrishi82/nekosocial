  ## Description-
 - nekoSocial is a social media app. Connect with people all around the world, Join now!

## Technologies Used -
 - React JS
 - Redux Toolkit
 - React Router V6
 - Vanilla CSS integrated with nekoUI Component Library


## Features -
- Create a Post
- Like a post
- Comment on a post
- Edit & Delete Post
- User Feed
- Sort by Date, Trending & Recent
- User Profile
- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Authentication
